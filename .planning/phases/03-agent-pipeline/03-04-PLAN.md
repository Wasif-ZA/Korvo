1. Update the tool_use handling block in `worker/lib/agent-loop.ts`:

In the `while` loop where `response.stop_reason === "tool_use"` is handled, update the block that iterates `response.content` to skip non-`tool_use` block types. Currently the code does:

```typescript
for (const block of response.content) {
  if (block.type !== "tool_use") continue;
  // ... call executeTool
}
```

This already correctly skips `server_tool_use` and `web_search_tool_result` blocks because they have `type !== "tool_use"`. However, the response.content may also contain these server blocks that need to be passed through to the assistant message. The current code already does `messages.push({ role: "assistant", content: response.content })` which includes all blocks. Verify this is correct.

The key change: when the response contains ONLY server tool blocks (no custom `tool_use` blocks), the `toolResults` array will be empty. In this case, do NOT push an empty `user` message with no tool results — just let the loop continue to the next API call where the server tool results are already in the assistant message.

Update the logic:

```typescript
if (response.stop_reason === "tool_use") {
  messages.push({ role: "assistant", content: response.content });

  const toolResults: Anthropic.ToolResultBlockParam[] = [];
  for (const block of response.content) {
    if (block.type !== "tool_use") continue; // Skip server_tool_use, text, web_search_tool_result
    let result = await executeTool(
      block.name,
      block.input as Record<string, unknown>,
    );
    result = filterBlockedUrls(result);
    toolResults.push({
      type: "tool_result",
      tool_use_id: block.id,
      content: result,
    });
  }

  // Only push tool results if there are custom tool calls to respond to.
  // Server tools (server_tool_use) are handled by the API and don't need
  // tool_result messages from us.
  if (toolResults.length > 0) {
    messages.push({ role: "user", content: toolResults });
  }
  steps++;
  continue;
}
```

  ): Promise<ContactResult[]> {

    const rawOutput = await runAgentLoop({
      systemPrompt: CONTACT_FINDER_SYSTEM_PROMPT,
      userMessage,
      tools: [

      maxSteps: 5,
      maxTokens: 2048,

    // Parse and validate JSON output
    const parsed = extractJsonArray(rawOutput);
    const validated = parsed.map(mapToContactResult);

    // D-03: Always return exactly 3

        name: "Unknown Contact",
        title: "Unknown",
        sourceUrl: null,
        confidence: "low",
        publicActivity: null,
      });
    }

    return validated.slice(0, 3);
  }
