# Agent Ownership

## Claude Code (Backend)
- API routes, server actions
- Database schema, migrations, RLS policies
- Claude API integration (prompt engineering, response parsing)
- BullMQ job definitions
- Supabase auth configuration

## Gemini CLI (Frontend)
- React components, pages, layouts
- Tailwind styling
- Client-side state management
- Form handling, input validation
- Copy-to-clipboard UX

## GitHub Copilot (Testing & DevOps)
- Unit tests, integration tests
- CI/CD pipeline
- Environment configuration
- Linting, formatting rules

## File Ownership Rules
- Never edit files owned by another agent without explicit handoff
- Shared files (types, schemas) require coordination
- .planning/ is maintained by Claude Code only
